Première app communautaire développée sur Nimiq, Nimipet voit s'ajouter tout une liste de nouvelles fonctionnalités. Pour rappel le jeu vous propose d’adopter une petite créature virtuelle dont vous devrez vous occuper afin de la maintenir en vie et de la faire progresser dans un classement. À noter que lorsque votre Nimipet meurt, 50% des NIM qui ont été minés pour le nourrir sont reversés aux Nimipets vivants avec un prenium pour ceux en vie depuis le plus longtemps (x3 contre x1 pour les plus jeunes).

Cette version a bénéficié d'une réécriture complète du code en Vue.js pour le front-end et en Laravel pour le back-end.

Voici les ajouts :

  • un nouveau système de nourriture avec des effets spéciaux sur votre Nimipet.
  • un nouvel item permettant à votre Nimipet d'hiberner (parfait pour l'hiver).
  • une paire de lunettes de soleil pour les Nimipets ayant atteint le rang #1 du classement.
  • un générateur d'avatars.
  • des Nimipets qui parlent et peuvent communiquer un message personnalisé aux visiteurs.
  • une traduction dans 10 langages dont le français.
  • un système de pages publics pour directement rediriger vers votre Nimipet, par exemple : https://nimipet.com/tipsy.

Ces ajouts sont loin d'être les derniers, en effet le développeur compte tokeniser les Nimipets (NIMI) sur la blockchain Ethereum pour permettre de les échanger à l'image des Cryptokitties. Outre de pouvoir les échanger les possesseurs de Nimipets seront également donné un score de "moralité" sur la façon dont ils s'occupent de leurs petits personages virtuels.

Outre cela, le développeur a également publié  le code en open source sur Github.
