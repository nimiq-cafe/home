Après un an d'existence les trois apps Nimiq voient leur première grosse mise à jour déployées sur le testnet.

Au niveau des nouvelles fonctionnalités dans l'interface on peut compter:

  • un nouveau processus de création de compte
  • la possibilité de stocker sa clé privée sous forme de fichier login (une image .png contenant un code QR ) dans le Keyguard.
  • l'ajout de codes QR pour les adresses et un scanner dans le Safe.
  • la possibilité de créer plusieurs adresses pour un même compte (et donc la possibilité de les stocker sous une même clé privée).

Mais la grosse nouveauté attendue est surtout celle du Gestionnaire de compte, une interface de programmation applicative (API) qui permettra à n'importe quel site ou application d'interagir directement avec votre compte Nimiq pour faire des paiements en quelques clicks depuis votre navigateur. Le package npm ainsi que le mode d'emploi de l'API sont disponibles ici.

N'hésitez pas à tester à vous rendre sur le Discord nimiq ou le topic officiel dédié pour partager vos retours d'expérience (en anglais).
