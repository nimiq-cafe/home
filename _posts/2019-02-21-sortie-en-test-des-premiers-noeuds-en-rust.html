Prise il y a plusieurs mois déjà, la décision de porter une partie du protocole sous Rust a atteint un premier objectif avec la sortie d'une premier version test des full nodes. Les développeurs intéressés sont encouragés à tester ces derniers, le code source et toutes les informations sont disponibles sur Github.

Pour rappel, le but de ce port de Nodejs vers Rust est d'améliorer les performances du protocole et de permettre un maintien de la base de code plus aisés sur le long terme. Bien sûr cette évolution n'aura pas d'impact sur la composante résolument web de Nimiq et son API javascript qui continuera d'être supportée et d'évoluer.

Mise à jour du 22/02 : les noeuds Rust sont officiellement sorties !
