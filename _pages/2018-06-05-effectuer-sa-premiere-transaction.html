Maintenant que vous disposez de vos premiers NIM nous allons voir comment les envoyer.

https://www.youtube.com/watch?v=04cJUryW6eg

Rendez vous sur le Safe puis cliquez sur la flèche au dessus de "Send".

Une fenêtre de transaction s'affiche avec les champs suivants :

  • SEND TO : rentrez l'adresse à laquelle vous voulez envoyer vos NIM
  • AMOUNT : le montant de NIM que vous désirez envoyer

Une fois rempli vous n'avez qu'à appuyer sur "SEND" pour terminer, néanmoins vous avez également la possibilité d'utiliser quelques options avancées en cliquant sur "ADVANCED SETTINGS".

  • MESSAGE : ce champs vous permet d'ajouter un message à votre transaction (maximum 64 caractères), attention néanmoins car ce message sera visible pour tous sur la blockchain.
  • FEE : ce curseur permet de régler les frais de transactions, plus les frais sont élevés plus la transaction sera traitée par les mineurs en priorité et donc plus elle sera rapide.
  • VALID FROM : cette option définit une fenêtre pendant laquelle la transaction pourra être traitée. En effet par défaut les transactions sur le réseau Nimiq ont environ 120 blocs pour être traitées et validées (2 heures environ), néanmoins vous pouvez réduire la fenêtre de validation. Par exemple si on veut que la transaction ne soit valable que pour les 30 prochains blocs ont utilisera :
    [Numéro du dernier bloc miné] - 120 + 30

Une fois ces paramètres définis on appuiera sur "SEND".

Une nouvelle "Nimiq Keyguard" s'ouvre dans laquelle vous devrez rentrer votre mot de passe puis cliquer sur "CONFIRM".

Voilà, votre transaction a été envoyée !

Ce guide du débutant se termine donc ici, vous êtes maintenant fin prêt à utiliser Nimiq.
